Introduction to Basic Python

What is Python?


Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

It is used for:

  • web development (server-side),
  • software development,
  • mathematics,
  • system scripting,
  • print "Hello world" to you.

Hello World

Let's write our first Python file, called helloworld.py, which can be done in any text editor.


>> print("Hello World")

Console should be send you back below.

Hello World

Congratulations! you're programmer now.


Variable


Variables are containers for storing data values.


Creating Variables

>> x = 5
>> y = "John"
>> print(x)
>> print(y)

x is store 5 to value (number)
and y is store John value (letter)

Console should be send you back below.

5 #value of x
John #value of y

Comments


There are 2 type of comments in Python

  • Inline comment we use : #comment
  • Multiple line comment we use : """ comment """
>> print('Not Comment')
>> #print('Inline comment')
>> """print('Multiple line comment') print('Multiple line comment')"""

Console should be send you back below.

print('Not Comment')

Data type


In programming, data type is an important concept. Variables can store data of different types, and different types can do different things. Python has the following data types built-in by default, in these categories:


  • Text Type: str
  • Numeric Types: int, float, complex
  • Sequence Types: list, tuple, range
  • Mapping Type: dict
  • Set Types: set, frozenset
  • Boolean Type: bool
  • Binary Types: bytes, bytearray, memoryview

we can check our data type by : type(x)

>> x = 3.14 #store data in variable
>> print(type(x))
float

After we store the data in variable and print type of variable Python will return type of data. In this case the return is float


Operators


Arithmetic operators are used with numeric values to perform common mathematical operations:


Operator Name Example
+ Addition x + y
- Subtraction x - y
* Multiplication x * y
/ Division x / y
% Modulus x % y
** Exponentiation x ** y
// Floor division x // y

let's try it!

>> x = 5
>> y = 2
>> print(x+y)
>> print(x-y)
>> print(x*y)
>> print(x/y)
>> print(x%y)

>> print(7)
